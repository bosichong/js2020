# 这是一个快速学习JavaScript的学习笔记

## <span id="m1">JavaScript 是脚本语言</span>


JavaScript 是一种轻量级的编程语言.
JavaScript 是可插入 HTML 页面的编程代码。
JavaScript 插入 HTML 页面后，可由所有的现代浏览器执行。
JavaScript 很容易学习。
