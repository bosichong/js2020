## <p id='m2'>JavaScript的代码写在哪里？</p>

`JavaScript`变量名区分大小写。

`;`用来分隔语句，如果语句是一行的情况下，可以省略（有些情况下是不行的）。

`HTML`代码中嵌入 `<script>  </script>`

外部引用：`<script src="./js/main.js"></script>`，放在`<head>`或`<body>`标签内。

## <p id='m2'>语法结构</p>

### 注释

```JavaScript
        // 单行注释

        /*
        *这是一段注释
        */

        /* 这也是一段注释 */
```

包括数值，字符串，布尔。

```JavaScript

null //空
```
